Lab 03 CSS Styling

For this lab, I added a hopping effect to the links on the page. To make it work, I set the links to display as inline blocks, which allows them to animate. From there, I applied an animation called bounce that plays over 0.9 seconds. Using keyframes, I defined the movement so the link jumps up, comes back down, does a smaller hop, and gradually settles back into place. I also styled a specific element using an ID selector by assigning it a unique ID in the HTML and targeting that ID in my CSS file.

Lab 05 Responsive Design

This module was about learning two ways to control layout in CSS — Flexbox and Grid. I used Flexbox for the navigation because it is great for lining things up in a single row, and Grid for the overall page structure because it lets you control both rows and columns at the same time. Figuring out when to use one over the other was the hardest part, but W3Schools helped me understand the difference. The main thing I learned is that Flexbox is best for small pieces of a page and Grid is best for the big picture layout.
